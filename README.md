# Matthew Green's Professional Site

Hello, this is my professional site. I have developed this site to showcase my skills I learned from recent courses in CMPA. This site includes links to my work, a slideshow on the homepage with images of homepages of some of my sites, and interactive features I also invite anyone to check my site out and offer feedback.

## Purpose

I created this professional site to showcase my skills to employers. It is a way to keep a collection of the work I have completed so far. I also invite anyone to take a look at my work to offer feedback and maybe even take away some ideas from my site. 

## Tools and Technologies

**GitHub** - I chose GitHub to host my site due to my familiarity and ease of use.

**VS Code** - I chose VS Code as my editor. I find this tool the best I have used so far during my coursework at TTU.
